# SatietyAI Course Resources

Centralized resource hub for all financial literacy course materials.

## Structure

- `core/` - Unbranded B&W masters (universal print versions)
- `teen/` - SatietyAI Teen Edition
- `young-adult/` - SatietyAI Young Adult Edition  
- `couples/` - SatietyAI Couples Edition
- `fsme/` - Financial Stability Made Easy (separate brand)

## URLs
